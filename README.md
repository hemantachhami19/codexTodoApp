## Run
```bash
# 1) unzip the project, then:

# 2) install deps
npm install

# 3) start server
npm start

# 4) open in browser
# http://localhost:3000
```

## Notes
- Todos are stored in `data/todos.json` so they persist between restarts.
- Frontend files are in `public/`.
